# 球星卡集市

一个微信小程序形态的球星卡交流网站原型，适合演示卡片浏览、求卡、发帖交流、私聊和个人卡册。

## 核心页面

- `pages/discover`：卡市首页，包含搜索、运动类型筛选、精选卡片和全站求卡清单。
- `pages/matches`：交流区，包含换卡、经验、活动等话题流。
- `pages/onboarding`：发布页，可发布换卡、出售、求卡或经验帖。
- `pages/chat`：卡片详情和模拟私聊。
- `pages/membership`：个人卡册、交易数据和求卡清单。

## 本地校验

```bash
npm test
```

当前版本使用 `services/api.js` 中的模拟数据，不依赖云函数即可预览前端体验。
